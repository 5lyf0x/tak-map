# Install / Upgrade

## Install

```bash
cd tak-map
sudo ./install.sh
```

Open TAK Map after install:

```text
https://<server-ip>:9444
```

## Upgrade from a release ZIP

```bash
cd ~/Downloads/tak/tak-map
unzip -o tak-map-v324.zip
cd tak-map
sudo ./install.sh
cd ~/Downloads
```

## Uninstall

```bash
cd tak-map
sudo ./uninstall.sh
```

## Notes

- The installer is intended for Raspberry Pi / Debian / Ubuntu-style systems.
- HTTPS is normally served through Nginx on port `9444`.
- The Python backend listens locally and is reverse-proxied by Nginx.
- Do not commit runtime certificates or local environment files generated by the installer.
